# 🚀 AWS S3 Static Website Deployment with Cloudflare & Custom Domain

## 📌 Project Overview
This project demonstrates deployment of a static website using AWS S3 with Cloudflare CDN and a custom domain. It includes DNS configuration, SSL setup, and global content delivery optimization.

---

## 🏗️ Architecture


User → Cloudflare (DNS + SSL + CDN) → AWS S3 Static Website


---

## 🌐 Live Demo
👉 https://sukhmaygiridevops.site

---

## ⚙️ Tech Stack

- AWS S3 (Static Hosting)
- Cloudflare (DNS + CDN + SSL)
- GoDaddy (Domain Registrar)
- HTML / CSS / JavaScript

---

## 🔧 Features

- Static website hosting on AWS S3
- Global CDN using Cloudflare
- Free SSL (HTTPS enabled)
- Custom domain integration
- DNS management via Cloudflare

---

## 🚀 Deployment Steps

1. Create AWS S3 bucket
2. Enable static website hosting
3. Upload website files (HTML/CSS/JS)
4. Configure Cloudflare DNS records
5. Connect domain via GoDaddy nameservers
6. Enable SSL (Flexible mode in Cloudflare)
7. Test live domain
